How to Install Heat and Devstack on EC2

Posted By: Tamir Korem on ,


Devstack is

  1. A great option to install and run an OpenStack cloud on your laptop or inside a VM on a cloud.
  2. Ideal for potential users who want to see what the Dashboard looks like from an admin or user perspective, and OpenStack contributors who wish to test their apps against a complete local environment.
  3. A well documented shell script that enables users to build complete Openstack development environments.

You can read more about Devstack here

Heat is

  1. A service to orchestrate multiple composite cloud applications using the AWS CloudFormation template format, through both an OpenStack-native ReST API and a CloudFormation-compatible Query API.
  2. A great tool that gives developers and administrators an easy way to create a collection of related Openstack resources and provision them in an orderly and predictable fashion.

You can read more about Heat here
