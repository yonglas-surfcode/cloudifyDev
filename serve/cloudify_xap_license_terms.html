The open source Cloudify software currently contains certain closed source components of GigaSpaces’ XAP software. As part of the license to the open source Cloudify software, and subject to compliance with the terms of such license, GigaSpaces grants you a non-exclusive, non-transferable, royalty-free license to these XAP components solely as incorporated into such Cloudify software. Please note that GigaSpaces intends to replace the XAP components of the Cloudify software with open source components of substantially the same functionality, at which time your license to the XAP components will expire and be replaced by a Apache 2.0 based license to the replacement open source components.
