How Cloudify Works

Cloudify is designed to bring any app to any cloud enabling enterprises, ISVs, and managed service providers alike to quickly benefit from the cloud automation and elasticity organizations today need. Cloudify helps you maximize application onboarding and automation by externally orchestrating the application deployment and runtime. Cloudify’s DevOps approach treats infrastructure as code, enabling you to describe deployment and post–deployment steps for any application through an external blueprint – AKA, a recipe, which you can then take from cloud to cloud, unchanged.

Cloudify's DevOps

Cloudify's DevOps approach enables you to install your application with a single shell or REST command, with no code changes. Just Script It

Cloud Migration

Cloudify simplifies cloud migration by provisioning the compute resources you need on demand, on the cloud of your choice (public cloud, private cloud, or even hybrid cloud), using its Cloud Driver Read about it in the Wiki.

Cloudify's Application Management

When it comes to easy application management, Cloudify auto-installs its agents on each VM to process your recipe.

PaaS & Cloud Infrastructure

Cloudify makes building your own PaaS a snap. Agents will process your recipe and install your application tiers, and with its tight integration with OpsCode's Chef, you can easily leverage any Chef cookbooks in the cloud deployment phase. Learn How Here.

Cloudify's & Tier Dependencies

Cloud Monitoring

Cloudify takes your cloud monitoring capabilities to the next level. You can easily monitor your application services for availability and performance, define any custom metrics you want to use, and ensure business continuity at all times. See the Code.

Elastic Cloud

With Cloudify the possibility of a truly elastic cloud is now possible. You can define scaling rules based on any custom metric - and Cloudify will scale out or scale in as needed.

Cloud Automation

When it comes to cloud automation - Cloudify ensures this isn't just a buzz word, but practically applicable in real time - providing the robustness your cloud needs. In case of any application service failure, Cloudify will allocate a new instance or a new machine - guaranteeing virtually no downtime or service interruptions.
